
# Â Hyve is a task management app that uses socket.io to interact between clients.

## Getting Started

First, make sure you have installed nodemon globally and mongodb on your computer:

From Root Folder:

### Server Side

First

```bash
cd server
```
then 

```bash
npm i
```
If you have nodemon installed
```bash
npm run dev
```
else
```bash
node index.js
```
Open another window on command line

From Root Folder:

### Client Side

First 

```bash
cd client
```
Then

```bash
npm i
```

Then

```bash
npm run dev
```

## You're all set!!

Now all you have to do is open [http://localhost:5173](http://localhost:5173) on your browser to see the result.
